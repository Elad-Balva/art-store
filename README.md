# Art Store

A full-stack e-commerce application for buying and selling art.

## Tech Stack

- **Frontend:** React, TypeScript, Axios, MUI, Tailwind CSS
- **Backend:** Express, TypeScript, PostgreSQL
- **Database:** PostgreSQL

## Project Structure

- **/client:** Frontend code
- **/server:** Backend code
- **/shared:** Shared types and interfaces

## Setup

### Prerequisites

- Node.js
- npm or yarn
- PostgreSQL
